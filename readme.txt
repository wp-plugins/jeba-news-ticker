== Description ==
This is Jeba news ticker wordpress plugin. By using a simple [jeba_ticker] shortcode easily can use Jeba news ticker in your site. In this plugin easily control scroll time and can use multiple ticker. And also scroll news from everywhere by using post_type.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After Activated the plugin use [jeba_ticker] shortcode where you want to use the Jeba news ticker.

Shortcodes

<strong>Use Jeba news ticker shortcode</strong>
<pre>[jeba_ticker]</pre>

<strong>Here can change</strong>
<pre>[jeba_ticker count="" post_type="" category="" id="" time="" title="" title_color="" title_bg=""]</pre>
